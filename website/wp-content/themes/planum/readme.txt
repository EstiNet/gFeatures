== Description ==

Planum is a Flat Design and Full-Screen Responsive Theme for publishers and entrepreneurs.

The Planum theme is easy to install and practical to use. Any color for the main elements, the background color for the main content using the built-in Customizer WordPress in the preview mode in real time. Theme Planum you will enjoy when you create a personal website, portfolio website, business or company.

The free version has limited features. More features and premium personal support will get by purchasing the Premium version Planum Theme.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.


== Quick Start ==

1. Choose your color, upload the logo image, go to Appearance > Customize

2. Theme supports 3 menus. Create Menu, go to Appearance > Menus

*Top Menu location placed in the hide/show top-bar (toggled button on Header)

3. Upload your image for Home Tagline, go to  Appearance > Header, and edit Home Tagline title, go to Appearance > Customize: Home Tagline section (height 750px, width 1200px)

*if you do not need a Home Tagline, go to Appearance > Customize: Display Options: Hide Header Image &Tagline

4. Define sticky posts (Posts > All Posts:  Quick Edit - check Make this post sticky) for show on Home

*Choose page as the front-page, go to Appearance > Customize: Static Front Page
*Want a background of the Home, large image, upload large image as a Featured Image (Page Edit) for selected page (to front-page)

5. Featured Image and Excerpt used posts and pages. To show Excerpt on the background featured image on the Header Post or Page - edit Excerpt (Page/Post Edit) And upload large image as a Featured Image (height 750px, width 1200px)

6. For tiled grid posts is required for each post upload image Featured image size of 800px on the width

Recommended Activate Jetpack Plugins Infinite Scroll Module (for index page posts tiles)

The theme includes Contextual Help.
